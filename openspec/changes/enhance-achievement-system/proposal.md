# 成就系统增强提案

## 问题背景

现有的成就系统已具备基本功能，但需要扩展以提升用户参与度和学习动力：

1. **成就类型有限**：当前只有8个默认成就，覆盖场景不够全面
2. **缺乏徽章系统**：用户完成成就后没有可视化的徽章展示
3. **缺少社交分享**：用户无法分享成就获得的喜悦
4. **成就等级体系**：没有区分普通、稀有、史诗等成就等级
5. **成就进度追踪**：用户无法查看解锁成就的进度

## 目标

扩展现有的成就系统，添加以下功能：

1. 增加更多成就类型和类别
2. 实现徽章系统（普通、稀有、史诗、传说）
3. 添加社交分享功能
4. 实现成就进度可视化
5. 添加成就解锁通知

## 范围

- **新增功能**：
  - 徽章等级体系（Common、Rare、Epic、Legendary）
  - 扩展成就类别（对话练习、评估、词汇学习、3D场景探索）
  - 社交分享API（生成分享图片、获取分享文案）
  - 成就进度追踪（显示完成进度百分比）
  - 实时成就解锁通知

- **修改功能**：
  - Achievement实体添加等级字段
  - gamificationService添加进度计算和分享功能
  - 前端成就展示页面优化

- **非目标**：
  - 不涉及用户系统修改
  - 不涉及课程系统修改

## 成功标准

1. 成就系统支持至少20个成就
2. 支持4种徽章等级
3. 提供社交分享功能
4. 显示成就解锁进度
5. 实时通知用户成就解锁
