# 3D场景扩展提案

## 问题背景

Lingua Journey 目前只有一个餐厅场景，学习场景的多样性不足，用户需要更多真实场景来练习语言技能：

1. **场景覆盖有限**：仅有餐厅场景，无法满足不同学习场景需求
2. **学习主题单一**：用户只能在餐厅环境中练习对话
3. **场景可复用性低**：没有统一的场景组件架构

## 目标

扩展3D场景数量，提供多样化的学习环境：

1. **咖啡馆场景** - 点咖啡、社交对话、休闲交流
2. **机场场景** - 值机、安检、登机、问路
3. **商店场景** - 购物、付款、退换货、询问

## 范围

### 新增场景

| 场景 | 主题 | 核心对话内容 |
|------|------|-------------|
| 咖啡馆 | 点单、社交 | 咖啡种类、点单用语、社交寒暄 |
| 机场 | 旅行、导航 | 值机、安检、登机、行李 |
| 商店 | 购物、交易 | 商品询问、试穿、付款、退换 |

### 技术范围

- 创建可复用的3D场景组件架构
- 实现场景选择器页面
- 添加场景间导航功能
- 复用现有的角色系统和语音交互

### 非目标

- 不涉及后端API修改
- 不涉及用户系统
- 不实现支付功能

## 成功标准

1. 至少新增3个可交互3D场景
2. 每个场景都有完整的角色和交互元素
3. 场景间可以自由切换
4. 用户可以在场景中学习相关词汇和对话
